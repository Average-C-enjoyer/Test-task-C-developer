# Data Handling Utility (C)

## Описание

Утилита объединяет данные из двух бинарных файлов `StatData`:

- записи объединяются по `id`
- `count` — суммируется
- `cost` — суммируется
- `primary` — становится 0, если хотя бы в одной записи 0
- `mode` — выбирается `max()`
- результат сортируется по `cost`
- сохраняется в выходной файл

В проекте также есть отдельная утилита тестирования.

---

## Сборка

Из корня проекта:

```bash
cmake -S . -B build
cmake --build build
```

---

## Запуск утилиты обработки

```bash
./build/handle_data <input1> <input2> <output>
```

Пример:

```bash
./build/handle_data dump1.bin dump2.bin result.bin
```

---

## Запуск тестов

```bash
./build/test_data
```

Покрываются следующие случаи:

* пустые массивы
* одинаковые `id` внутри одного файла
* 100000 элементов (stress test)
* ошибки чтения
* повреждённый файл
* логика `primary = 0`
* `mode = max()`
* одинаковые `cost`
* отрицательные `cost`
* уже отсортированный массив
* полностью одинаковые `id`
